# KarityLab-Excel-HR-Data-Analysis-Case-Study
This repository contains an HR data analysis case study designed to practice data cleaning, applying company rules, and preparing employee data for decision-making. It includes exercises on formatting, validation, performance evaluation, error handling, and reporting, providing a realistic scenario for aspiring data analysts.

## 🎯 Objectives
- Clean and standardize employee names.
- Ensure email addresses follow a consistent lowercase format.
- Apply company performance rules:
  - Score ≥ 80 → Outstanding
  - Score ≥ 60 → Good
  - Score ≥ 50 → Average
  - Score < 50 → Poor
- Flag employees who did not meet the pass mark.
- Identify promotion eligibility (Score ≥ 60 **and** bonus received).
- Handle missing or invalid records gracefully.
- Summarize employees in the format:  
  `Name | Department | Performance Rating`
- Output top performers in the format:  
  `Name - Rating`
